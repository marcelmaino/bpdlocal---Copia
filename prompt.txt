

-Sistema de Dashboard para Controle de Eficiência e Gestão Financeira de Poker



-- Visão Geral do Sistema

Desenvolver um dashboard web completo para gerenciamento de performance e controle financeiro de jogadores, agentes e clubes de poker. O sistema terá interface responsiva com dois níveis de acesso distintos e visualização de dados em tempo real. Terá gráficos e análises por ia para complementar 
Vamos utilizar docker para desenvolvimento local e produção.





-- Tipos de Usuários e Autenticação

 Admin (Administrador) 
Acesso completo a todos os dados 

Visualização de todos os jogadores, agentes e clubes 

Capacidade de aplicar todos os filtros disponíveis 

Acesso a relatórios consolidados 

login:admin, senha: adm121456
este usuário pode ser incluido como padrão 


Jogador
Acesso restrito apenas aos seus próprios dados
Login usando a coluna "playerName" da base de dados
Senha padrão: Primeira letra do nome em maiúscula + "2025" + últimos 4 caracteres do playerName
Exemplo: João Silva → login: "João Silva", senha: "J2025ilva"
Visualização limitada aos seus resultados pessoais;
fazer o select e agrupar informações de um único usuário por login;

Script para a criação de uma tabela de usuários distinta com os dados de login e senha padrão com base nos registros da coluna playerName.


utulizar auth.js para facilitar

- - Tabela de Registros Completa Para o frontend das tabelas utilizar o componente datatables (datatables.net) Funcionalidades da tabela: Exibe todas as colunas menos a "linha_id" Recursos avançados: Ordenação por qualquer coluna (clique no header) Busca dentro da tabela Exportar para PDF OU EXCEL dentro do período selecionado Paginação com opções: 10, 50, 100, 1000, "Todos" Altura da tabela ajustável dinamicamente Seleção de colunas visíveis Agrupamento por jogador/clube/agente Totalizadores no rodapé utilizar as opções do datatable para aplicar estas funcionalidades.  



Página Quadro de Desempenho Geral Container com abas para diferentes visualizações largura 100%: Aba 1: Gráficos Gráfico de linha: Evolução do balanço ao longo do tempo Gráfico de barras: Comparação de ganhos vs taxas por período Gráfico de pizza: Distribuição de mãos por clube Gráfico de área: Volume de jogos por dia da semana Geração de gráficos personalisados Aba 2: Métricas Taxa de vitória (win rate) ROI (Return on Investment) Média de ganhos por mão Total de horas jogadas (se disponível)

Fluxo de Dados
Login: Usuário acessa com playerName + senha padrão
Carregamento inicial: Dashboard carrega dados da semana atual
Aplicação de filtros: Atualização em tempo real dos cards e gráficos
Interações: Cliques em gráficos mostram detalhes específicos
Exportação: Dados podem ser baixados em formatos diversos


Responsividade
Desktop: Layout completo com todos os elementos visíveis
Tablet: Cards em grid 2x2, gráficos empilhados
Mobile: Cards em coluna única, menu hambúrguer para filtros


Performance Lazy loading para a tabela de registros   Este dashboard proporcionará uma visão completa e intuitiva dos dados de poker, permitindo análises detalhadas e tomada de decisão baseada em dados para todos os envolvidos no ecossistema.
